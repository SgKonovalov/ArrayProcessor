Консольное приложение написано на Java 1.8.
Включает 2 класса:
1) Element - содержит координаты матрицы -  атрибуты  int x, int y (private поля с геттерами/сеттерами) и конструкторы для инициализации объектов: с параметрами и без.
2) Main - содержит методы: 
1. main();
2. fillTheArray(Element[][] sourceArray) - для заполнения многомерно массива. Принимает в качетсве параметра статитеческое поле класса Main - сам многомерный массив;
3.  findTheNeighbors(Element element) - получает на вход объект класса Element и возвращает список «соседних» экземпляров их матрицы.
